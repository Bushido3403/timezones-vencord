# Timezones

A Vencord plugin that allows you to track and display other users' local times directly in messages.

## Features

- **Message Timestamps**: See each user's local time next to their messages
- **Timezone Management**: Set and manage timezones for any user via right-click context menu
- **Curated Timezone List**: Choose from a practical selection of major cities worldwide
- **Customizable Display**: Toggle between 12/24-hour format and GMT offset display
- **Persistent Storage**: User timezone preferences are saved locally

## Usage

### Setting a User's Timezone

1. Right-click on any user
2. Navigate to **Timezones** → **Set Timezone** (or **Change Timezone** if already set)
3. Select a region (Americas, Europe, Asia, Oceania, Africa)
4. Choose a city from the list (for their timezone, for example "Los Angeles" for PST)

### Viewing Local Times

Once a timezone is set, their local time will appear next to their messages as a small tag (e.g., "7:30 PM Local"). Hover over the tag to see the full date and timezone identifier.

### Removing a Timezone

1. Right-click on the user
2. Navigate to **Timezones** → **Remove Timezone**

## Settings

- **24 hour time**: Toggle between 12-hour and 24-hour time format
- **Show local time next to messages**: Enable/disable the inline time display
- **Show GMT offset**: Display the GMT offset (e.g., GMT+2) alongside the time

## Available Timezones

The plugin includes major cities from:
- **Americas**: New York, Los Angeles, Chicago, Toronto, São Paulo, and more
- **Europe**: London, Paris, Berlin, Moscow, and more
- **Asia**: Tokyo, Singapore, Dubai, Mumbai, and more
- **Oceania**: Sydney, Melbourne, Auckland, and more
- **Africa**: Cairo, Johannesburg, Lagos, Nairobi
